# 🛠 SQL Server DBA Backup & Recovery Project

## 📌 Overview
This project demonstrates a complete production-level SQL Server backup and recovery system.

## 🚀 Features
- Full Backup (Daily)
- Differential Backup (Every 4 Hours)
- Transaction Log Backup (Every 15 Minutes)
- SQL Server Agent Job Automation
- Backup Cleanup (Retention Policy - 7 Days)
- Email Alert System for Job Failures
- Disaster Recovery Simulation
- Point-in-Time Recovery using Timeline

## 🧠 Concepts Covered
- Backup Chain (Full → Differential → Log)
- Log Sequence Number (LSN)
- Recovery Models (FULL)
- Restore Sequence and Automation
- Maintenance Plans

## ⚙️ Tools Used
- Microsoft SQL Server
- SQL Server Management Studio (SSMS)

## 📂 Project Structure
- Database Setup Scripts
- Backup Scripts
- SQL Agent Job Configuration
- Cleanup Strategy
- Restore Process Documentation

## 🎯 Outcome
Successfully designed and implemented a real-world DBA backup and recovery system with automation and disaster recovery testing.
